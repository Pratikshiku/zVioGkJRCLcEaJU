## Gansu Intangible Cultural Heritage Website

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/271337697ebf4883b5eac163837773fa/ghb20250917) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript` etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Function Description
The system is divided into (roles): Regular User, Administrator.

**Regular User Functions:**
1. Login
2. Register
3. Homepage
4. Product Information, Favorite Products, Add to Cart, Buy Now, Comment on Products,
5. Inheritor Display
6. Intangible Cultural Heritage Exhibition
7. Personal Center
8. My Orders
9. My Addresses
10. My Favorites
11. Shopping Cart
12. Customer Service Information
13. Change Password
14. Personal Information
15. My Product Review Management
16. Application Management
17. My Favorites Management
18. My Order Management (Shipped Orders, Unpaid Orders, Cancelled Orders, Paid Orders, Refunded Orders, Completed Orders)


**Administrator Functions:**
1. Login
2. Change Password
3. User Management
4. Product Category Management
5. Application Management
6. Product Information Management
7. Product Review Management
8. Carousel Management
9. Intangible Cultural Heritage Exhibition
10. Customer Service Management
11. Order Management (Shipped Orders, Unpaid Orders, Cancelled Orders, Paid Orders, Refunded Orders, Completed Orders)
